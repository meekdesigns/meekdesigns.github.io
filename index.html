About Me

Hello there! I'm a front-end developer and aspiring UX designer based in Nairobi.

Having spent a large chunk of my life in computer school, I've finally found my passion at the crossroads of design and technology. I started off as an undergraduate in Computer Science, then worked some graphic design gigs, and later went back to school for web design and new media. The one lesson that stood out throughout this journey was the importance of the end user. Whether a physical or digital product, its purpose needs to be obvious, and the product itself has to be easy to use. So no matter what I'm working on, I always have the user in mind and try my hardest to create an enjoyable experience for them.

Currently, I spend most of my time designing websites and marketing material for small businesses and participating in Sololearn challenges to keep my skills sharp. When I'm not busy, you can find me obsessing over Deejaying, or working on one of my many hobbies like playing piano. I tend to switch things up every few months, so if you want to know what I'm up to, just follow me! :)

Project Details

Meek Designs

Responsive Web Design

This tribute page was created for a web design project. The goal was to make a website to grow my own brand. I took this chance to practice using the Bootstrap 3 framework to create a tribute to my works, Meek Designs. Lucky for me, Meek Designs has such strong branding that the page nearly designed itself!

I kept the color palette simple with black, white, and the signature TOP shade of blue-green for the accent color. To add some visual interest, I used a subtle twill texture for the blue backgrounds and styled the headings with a white and black background to give it a label maker tape effect.

Technologies Used:

Bootstrap 3 Responsive Design HTML5 CSS3 JavaScript jQuery

Project Details

Jokimba Suppliers

Responsive Web Design

Jokimba Suppliers (JS) is a side project of stationery distributor Josephat Mwangangi, where he challenges himself to be among the leading stationery and office equipment distributors in Kenya. The purpose of the JS website is to host his collection of stationery equipment, which were originally shared as Instagram stories.

Josephat really liked the bold, high-contrast look of meekdesigns.com, a web design website, so that's what inspired the design of this project. The color palette is mainly black and white with splashes of magenta and neon green from the JS brand. It was built with the latest version of Bootstrap 4 and uses a small JavaScript library, to prevent the animated GIFs from autoplaying when the page loads.

Technologies Used:

Bootstrap 4 Responsive Design HTML5 CSS3 jQuery WordPress

Project Details

Dantech Technologies

Responsive Web Design

This project was for Daniel Muia, a developer and online cyber C.E.O, based in Kisii-KENYA. His main goal was to attract prospective clients for online cyber services, so he wanted a website that showcases some of his best work.

I kept the design of this website simple to help Daniel's work stand out. His reel (AKA a video portfolio) is the most important piece of content, probably the first thing his future clients would look for, so it is placed front and center on the landing page. To make the embedded videos responsive and looking good on all devices, I used a jQuery plugin called FitVids, which gives the videos a fluid width. For the contact form, I used PHPMailer, an email sending library for PHP, to make it functional and also coded in a "honeypot" to help prevent spam. Other than the two plugins I just mentioned, the rest of this website was completely hand-coded.

Technologies Used:

Responsive Design HTML5 CSS3 PHP PHPMailer WordPress
